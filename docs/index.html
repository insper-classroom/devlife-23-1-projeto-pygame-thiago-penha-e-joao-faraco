Descrição do Jogo

Four Seasons Odyssey: Journey Through Nature é um jogo de plataforma que leva o jogador por uma aventura emocionante e desafiadora através de quatro estações diferentes. Inspirado no clássico Super Mario, o jogo apresenta uma jogabilidade simples e intuitiva que é fácil de aprender, mas difícil de dominar. Cada estação traz um conjunto único de desafios, com diferentes tipos de monstros e obstáculos para superar. Com gráficos coloridos e uma trilha sonora vibrante, Four Seasons Odyssey: Journey Through Nature é um jogo envolvente e divertido para jogadores de todas as idades que desejam desfrutar de uma aventura emocionante e desafiadora enquanto exploram as maravilhas da natureza.

Instruções

Após baixar o pacote zip, é necessário fazer a instalação do python e do pygame, caso o dispositivo não tenha esses recursos. Para baixar o python vá até https://www.python.org/downloads/ e baixe a versão mais adequada para seu sistema operacional. Após isso é necessário instalar o pacote pygame, em seu terminal digite python -m pip install pygame. O próximo passo da instalação é extrair o arquivo zip. No terminal é necessário acessar o diretório em que o jogo está portanto será necessário digitar cd "C:\Users\seu_nome_de_usuário\Downloads\devlife-23-1-projeto-pygame-thiago-penha-e-joao-faraco-main ", depois digite python jogo.py ou python 3 jogo.py dependendo da sua máquina.
